# Writer Tool Skill

## Role
Generate viral short-form scripts and platform metadata per niche tone.

## Output JSON fields
- script: 150-300 words, spoken style
- title: max 100 characters
- description: include 3-5 hashtags
- tags: 5-10 YouTube tags
- tweet: max 280 characters
- tiktok_caption: short hook + hashtags
- keywords: 3-5 words for stock footage search

## Tone by niche
- football: passionate, opinionated hot takes
- movies: witty, entertaining reviews
- anime: expressive fan reactions
- crypto: urgent, analytical market commentary
